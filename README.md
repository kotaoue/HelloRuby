## Installing Ruby

Ruby 3.2.3 is required. Install it using [rbenv](https://github.com/rbenv/rbenv) or [RVM](https://rvm.io/).

**Using rbenv:**

```bash
rbenv install 3.2.3
rbenv local 3.2.3
```

**Using RVM:**

```bash
rvm install 3.2.3
rvm use 3.2.3
```

Verify your Ruby version:

```bash
ruby -v
# ruby 3.2.3 ...
```
